### Task 2: agent/settings.py Module

Five functions implemented:

| Function | Purpose |
|----------|---------|
| `_derive_fernet_key()` | sha256(hostname:user) → urlsafe_b64encode → `Fernet` instance |
| `load_settings_json()` | Read settings.json → dict; silently returns `{}` on FileNotFoundError/JSONDecodeError/OSError |
| `save_settings_json(data)` | Atomic write via `.tmp` + `os.replace()` |
| `encrypt_api_key(plaintext)` | Fernet.encrypt → base64 string for settings.json storage |
| `decrypt_api_key(blob)` | Fernet.decrypt → plaintext or `None` (bare `except Exception`) |

## Mutability Mode Determined

**MUTABILITY_MODE: direct field assignment**

`config.provider = "anthropic"` works without raising `ValidationError`. The current `Settings` class uses `SettingsConfigDict(...)` without `frozen=True`. Plan 03 POST handler can use:

```python
config.provider = stored["provider"]
config.ollama_model = stored.get("ollama_model", config.ollama_model)
```

No `object.__setattr__` required.
